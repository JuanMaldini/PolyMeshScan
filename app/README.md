# app/

App iOS de captura en tiempo real. Punto de partida: fork/customización de
[RTAB-Map](https://github.com/introlab/rtabmap) (`app/ios/RTABMapApp`), no un capturador ARKit
desde cero. Ver `../docs/PLAN.md` secciones 5-7 para el porqué y las dos técnicas de captura
consideradas (malla densa vs RoomPlan).
